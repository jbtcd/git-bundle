## Installation

To install, use composer:

```bash
$ composer require jbtcd/git-bundle
```

## Testing

``` bash
$ composer run tests
```

## Note

This plugin makes Symfony significantly slower. Approximately 10 ms per commit to be read. By default, this means about 100 ms.
